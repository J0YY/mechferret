# Applied Use Case: Computational Biology Literature Triage

A compelling applied autoresearch use case is computational biology literature
triage. The agent can search paper abstracts, extract mechanistic claims, track
which experiments support each claim, and identify contradictions across model
organisms, assays, or datasets.

The same architecture also applies to legal research and coding research, but
biology makes evidence quality visible because claims often depend on methods,
sample sizes, and reproducibility.

The system should avoid pretending that a citation proves a claim. It should
separate extracted claims, source metadata, confidence, and unresolved questions.
