# Autoresearch Hackathon Brief

The hackathon rewards systems that enable agents to iteratively plan, search,
and synthesize information over extended horizons. Strong entries should show
agent architectures and control loops, retrieval and knowledge synthesis, or an
applied autonomous research agent in a high-impact domain such as law,
computational biology, or coding.

Judges are likely to value visible systems work over a single prompt wrapper.
An autoresearch project should expose how it plans, how it selects sources, how
it validates claims, and how it decides that another search round is necessary.

The strongest demo artifact is an auditable research dossier: a short answer,
a claim graph, a citation ledger, contradictions, unresolved gaps, and an
inspectable trace. This makes the system legible to judges in a short demo slot.
