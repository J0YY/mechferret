# MechFerret Project Instructions

Build for the Autoresearch Systems Hackathon by optimizing for visible,
inspectable research autonomy rather than opaque chatbot output.

Non-negotiables:

- A run must produce citations, confidence scores, source diversity, gaps, and
  contradictions.
- The same corpus and question must produce replayable artifacts.
- Offline demo mode must work with no API keys.
- Optional OpenAI, Modal, and Raindrop integrations must not break the offline
  path.
- Borrow only high-level architectural ideas from the referenced Claude Code
  tree: tool/task registries, persistent memory, selective recall, traceability,
  and critic loops. Do not copy leaked source implementation.
