# Changelog

All notable project changes should be recorded here. This project follows
semantic versioning once releases are cut.

## 0.1.0 - Unreleased

### Added

- Offline-first research loop with source ingestion, evidence memory, tracing,
  reports, paper generation, review, audit, verification, and bundle tooling.
- CLI command index, generated CLI reference, examples, shell completion, and
  script-friendly JSON payloads.
- OpenVLA SAE scaffold, template packaging, and smoke checks for local project
  initialization.
- CI gates for tests, bytecode compilation, doctor checks, JSON smoke tests,
  release hygiene, wheel metadata, packaged runtime assets, and installed
  console scripts.
- Community health files: contributing guide, support policy, security policy,
  code of conduct, issue templates, pull request template, license, and citation
  metadata.

### Notes

- Core operation remains offline by default; provider APIs, Modal, cluster
  execution, and real model backends are optional integrations.
